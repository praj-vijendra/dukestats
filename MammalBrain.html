Lets create a multiple linear regresson model of log brain weight on log body, log gestation and log litter.

regbrain = lm(logbrain ~ logbody + loggestation + loglitter, data= sub.mammalbrain)

We can interpret the model by: Body Weight: If there is a 10% increase in the body weight, then multiplicative factor associated with median brain weight is 1.05634 (1.1^0.57507). The 95% confidence interval for the mutiplicative factor is between 1.0498439 and 1.0628764.

Gestatiton: If there is a 10% increase in the gestation, then multiplicative factor associated with median brain weight is 1.040638 (1.1^0.41794). The 95% confidence interval for the mutiplicative factor is between 1.0132721 and 1.0687433.

Litter: If there is a 10% increase in the litter, then multiplicative factor associated with median brain weight is 0.9708796 (1.1^-0.31007). The 95% confidence interval for the mutiplicative factor is between 0.9498062 and 0.9924203.
