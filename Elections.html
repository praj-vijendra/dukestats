Lets create the scatter plot for County and Buchanan. We can see from the scatter plot that the value for Palm beach is an outlier.

ggplot(data = elections) + geom_point(mapping = aes(x =  Buchanan2000, y = County))

Lets create the scatter plot for Bush and Buchanan. We can see from here aswell that the value for Palm beach is an outlier.

ggplot(data = elections) + geom_point(mapping = aes(x =  Buchanan2000, y = Bush2000))

We can create a subset of the data by removing the outlier to create the linear regression model without the outlier.

elections2 = subset(elections, elections$County != 'Palm Beach')

ggplot(data = elections2) + geom_point(mapping = aes(x =  Buchanan2000, y = Bush2000))

election_model = lm(Buchanan2000 ~ Bush2000, data = elections2)

plot(y = election_model$residual, x=elections2$Buchanan2000, ylab = "Residuals", xlab = "Buchanan2000", main = "Plot of residuals versus Buchanan")
abline(0,0)

qqnorm(election_model$residuals, ylab = "Residuals", main = "Normal quantile plot of residuals")

We can see from the scatter plot that the relationship between Bush and Buchanan is not perfectly linear. Also, plot of residual versus Buchanan does not exhibit constant variance. Looking at the qq plot, we can see that the plot does not exhitbit normal distribution.

We can take the log values of Bush and Buchnanan and try to see if these variable have a linear relationship.

elections2$LogBuchanan2000 = log(elections2$Buchanan2000)

elections2$LogBush2000 = log(elections2$Bush2000)

election_model2 = lm(LogBuchanan2000 ~ LogBush2000, data = elections2)
